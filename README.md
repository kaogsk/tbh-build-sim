# 🗡️ Taskbar Hero — Save Inspector & Stat Simulator

> Ferramenta criada entre amigos para facilitar a vida na hora de montar o personagem perfeito em **Taskbar Hero**.

---

## 💡 O que é isso?

Sabe quando você tá com um item incrível em mãos mas não sabe qual pedra (socket) colocar nele pra tirar o máximo de performance? É pra isso que essa ferramenta existe.

Você carrega o arquivo de save do seu jogo, e ela mostra **tudo sobre seu personagem** — atributos, equipamentos, resistências, chances de esquiva — e ainda te deixa **simular** como ficariam suas fichas se você trocasse as pedras dos seus itens, **antes mesmo de gastar os recursos no jogo**.

---

## ✨ O que a ferramenta faz

- 📂 **Carrega seu save** — basta fazer upload do arquivo de save do jogo
- 🧙 **Mostra todos os status detalhados** do seu personagem (ataque, defesa, HP, velocidade, resistências, esquiva, ouro, exp...)
- 🎒 **Exibe seus equipamentos** com ícones reais do jogo e tooltip com os atributos de cada item
- 💎 **Simula a troca de pedras** — clique em qualquer socket e veja como seus stats mudam em tempo real
- ⚔️ **Compara original vs simulado** lado a lado, mostrando a diferença exata de cada status
- 🌐 **Suporte a inglês e português** na interface

---

## 🎮 Como usar

1. **Abra a ferramenta** pelo link
2. Clique em **"Load Save File"** e selecione o arquivo de save do seu jogo
3. Veja todos os seus equipamentos e status carregados na tela
4. Clique em qualquer **socket (círculo vazio ou com pedra)** num item
5. Escolha uma pedra diferente e veja os **stats mudarem na hora** na coluna "Simulated"
6. Decida se vale a pena trocar — sem gastar nada no jogo!

---

## 🔍 Como encontrar o arquivo de save

O arquivo de save do **Taskbar Hero** fica neste caminho (Windows):

```
C:\Users\SEU_USUARIO\AppData\LocalLow\TesseractStudio\TaskbarHero\SaveFile_Live.es3
```

Substitua `SEU_USUARIO` pelo seu nome de usuário do Windows. É esse arquivo `.es3` que você vai carregar na ferramenta.

## 👥 Sobre o projeto

Essa ferramenta foi feita **entre amigos, pra amigos**. A ideia surgiu na conversa do grupo quando a gente ficava perdendo pedras raras tentando adivinhar qual combo ficaria melhor nos itens ideais.

Nenhum dado é enviado pra nenhum servidor — tudo roda direto no seu navegador. Seu save fica só na sua máquina. 🔒

---

## 📸 Interface

A ferramenta tem:
- Painel esquerdo com os equipamentos do personagem
- Painel direito com todos os status (Original | Diferença | Simulado)
- Tooltips com informações completas de cada item ao passar o mouse
- Simulador de pedras com preview em tempo real

---

## 🛠️ Tecnologias

Feito com HTML, CSS e JavaScript puro + Vite. Sem banco de dados, sem conta, sem login. É só abrir e usar.
